# Calculadora de IMC

Um projeto criado com fins educativos e para treino de linguagem. Consiste em uma página de calculadora de IMC com campos diferentes e interface completa.

## Objetivos
* Prática da estrutura e funcionamento do JavaScript.
* Prática da vinculação entre linguagens diferentes, HTML e JavaScript

## Tecnologias
* Linguagem: JavaScript, HTML e CSS.

## Instalação e Uso
1. Clone o repositório ou baixe os arquivos.
2. O projeto não exige instalações externas, basta apenas executar o arquivo "index.html".
